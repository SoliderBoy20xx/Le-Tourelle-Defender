Features
-Fixed-position defensive turret controlled by the player.
-Primary weapon with high fire rate.
-Secondary weapon with long cooldown and high damage.
-Base defense system

-Wave-based enemy spawning system.
-Two enemy types:
  Fast enemy (low health, high speed)
  Juggernaut enemy (high health, low speed)

Architecture
-ScriptableObjects used for gameplay data.
-Object Pooling system implemented for projectiles and enemies.
-Reusable pooling architecture using custom PoolManager instead of unity's pool system
-WaveManager using configurable spawn points and wave definitions.



Controls
Q / D : Rotate turret
Left Click : Primary fire
Right Click : Secondary fire




Unity Version
Unity 6
